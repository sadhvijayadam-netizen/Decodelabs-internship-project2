# 🌸 Iris Flower Classification using Artificial Intelligence

## 📖 Description
This project demonstrates a Machine Learning classification model using the Iris Flower Dataset. The model is trained to classify iris flowers into three species: Setosa, Versicolor, and Virginica based on sepal and petal measurements.

## 🎯 Project Objective
To build an Artificial Intelligence classification model that accurately predicts iris flower species using supervised learning techniques.

## ✨ Features
- 📊 Dataset Loading and Analysis
- 🧹 Data Preprocessing
- 🔀 Training and Testing Data Split
- 🌸 Flower Species Classification
- 📈 Model Performance Evaluation
- 📋 Confusion Matrix Generation
- 📑 Classification Report
- 🎯 Accuracy Measurement

## 🛠️ Technologies Used
- 🐍 Python
- 🐼 Pandas
- 🤖 Scikit-learn

## 📂 Dataset Information
The Iris Dataset contains 150 flower samples with the following features:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## 🌺 Classification Categories
- Setosa
- Versicolor
- Virginica

## ⚙️ Project Workflow
1. 📥 Load the Iris Dataset
2. 📊 Analyze Dataset Information
3. 🔀 Split Data into Training and Testing Sets
4. 🤖 Train the Classification Model
5. 🎯 Predict Flower Species
6. 📈 Evaluate Model Performance
7. 📋 Display Accuracy and Confusion Matrix

## 📏 Performance Metrics
- Accuracy Score
- Precision
- Recall
- F1-Score
- Confusion Matrix

## 🚀 Expected Outcome
The model successfully classifies iris flowers into their respective species and achieves high prediction accuracy using machine learning techniques.

This project demonstrates the fundamentals of Artificial Intelligence and Machine Learning by building a flower classification system capable of accurately identifying iris species based on flower measurements.
